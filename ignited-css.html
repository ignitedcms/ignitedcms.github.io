Introduction

Kamijs is a minimalistic CSS/JS framework with a sprinkling of Vue on the front end.


It is designed to be straight forward and easy to use, and most importantly framework agnostic. The code is yours to own, and MIT licensed, meaning you can use it freely in your next project.

Roadmap and known issues

Kamijs is almost ready for its first release candidate v.1.0.0. However, there are a few fixes that need adressing

  • Spinner as component and thickness color prop
  • Make radio and checkbox more flexible, options as components
  • Create more examples for input types add and name props
  • Document typography components,h1,h2 paragraph blockquote
  • Make sure dropdown menu, and menubar support touch devices on android
  • Create and finalise 'command' and 'calendar' and 'context menu' components
  • Add datepicker support for disabled dates, ranges, range pickers
  • Ensure combobox scrolls down when keyboard navigating
  • Fix no scrolling when 'sheet' component is open
  • Tidy main menu and mobile menu and ship with better accessibility
