Can you explain how to create a responsive layout with Tailwind CSS?

Creating a responsive layout with Tailwind CSS is straightforward using its built-in responsive modifiers. Here's how you can do it:

Tailwind uses a mobile-first approach, which means:

  • sm: - applies to screens 640px and wider
  • md: - applies to screens 768px and wider
  • lg: - applies to screens 1024px and wider
  • xl: - applies to screens 1280px and wider
  • 2xl: - applies to screens 1536px and wider

For example, to create a layout that's one column on mobile and two columns on medium screens:

<div class="grid grid-cols-1 md:grid-cols-2 gap-4"> <div class="bg-white p-4 rounded shadow">Item 1</div> <div class="bg-white p-4 rounded shadow">Item 2</div> </div>

You can also change other properties based on screen size, like text size, padding, or visibility.
