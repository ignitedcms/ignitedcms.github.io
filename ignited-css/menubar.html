Menubar

A visually persistent menu common in desktop applications that provides quick access to a consistent set of commands. This is fully keyboard accessible and ARIA compliant. It also has support for pseudo radio buttons and checkboxes.

Useful tip
Menubars don't have a great user experience on mobile portrait or landscape views. Consider swapping it out for the 'mobile menu' component on smaller screens
