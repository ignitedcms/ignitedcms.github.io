Dropdowns

Displays a menu to the user — such as a set of actions or functions — triggered by a button, which is fully keyboard accessible. Note this also has support for pseudo radio buttons and checkboxes.

Useful tip
Heavily nested dropdowns don't have a great user experience on mobile portrait or landscape views. Consider swapping it out for the 'mobile menu' component on smaller screens

Positioning

You can easily customize the position of the dropdown menu via the position, align, offset and gap props
Dropdown API Prop Options position top, bottom, left, right align start, center, end offset value in pixels (eg. -10px) gap value in pixels (eg. 10px)

Note this requires javascript (Vuejs 2.0)
