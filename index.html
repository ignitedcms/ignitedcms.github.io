Create bespoke Laravel frontend applications in record time
IgnitedCMS is a content management system for building frontend customer facing websites. It comes with support for testing, modern tooling, an ecosystem of official packages, and more. All available with Laravel
Go to docs Github 
composer create-project laravel/laravel example-app
cd example-app
composer require ignitedcms/ignitedcms

FAQ

IgnitedCMS is a micro content management system built on a Laravel 11 core, its aim is simple. To easily allow you to create a frontend customer facing website with SEO friendly routing etc. It can drop right into your existing Laravel Project.

Yes, we built this with the community in mind. It is our promise to you that this will always remain open to the public

Currently this is in development, we aim to ship a beta version soon and it will be available via composer. Version 1.0.0

Let the site generate all your routes for you. The engine uses the unique way in which you create content and section types to intelligently generate all routing for you. Taking away any headaches and offering SEO friendly routing for web crawlers.

I was often dis-satisfied with headless CMS options, slow loading pages, confusing route setups and limited features requiring a paid option so IgnitedCMS was born.

IgnitedCMS is not yet another CMS. Instead, we have written IgnitedCMS from the ground up on top of Laravel to be simple, elegant, and opinionated.
